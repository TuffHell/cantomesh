平仄工具台

为撰曲而生的四件工具：校验平仄、粤拼标注、韵脚检索、句式模板。 二万余字粤拼库，全在浏览器内离线运算——规则透明，即是可审计的非遗诚信。

平仄校验 Verify

按行或标点分句。上句（单数）句末须仄；下句（双数）句末须平且押同一韵。

⌘/Ctrl + Enter 平阴平/阳平 · 仄上去入声

粤拼标注 Annotate

粘贴任意中文（简繁皆可），逐字标注粤拼、声调与平仄，并显示标音覆盖率。

韵脚检索 Rhyme

输入一个汉字或粤拼（如 家 或 gaa1），检索同韵母组的字，按平/仄分列。

句式模板 Template

选择平仄模板，逐位比对你的句子是否合律。

跨代生成 Translate

将现代白话 / 流行歌词，经 Claude 起草 → 平仄校验 → 违规重写 的神经-符号回路生成唱词。

生成需调用大模型，密钥不能置于公开页面，故运行于 FastAPI 后端。 本工具台为公开静态站，专注离线分析四件套。本地启用： uvicorn app.main:app --reload 并设置 ANTHROPIC_API_KEY。
