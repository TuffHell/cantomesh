平仄工具台

為撰曲而生的四件工具：校驗平仄、粵拼標註、韻腳檢索、句式模板。 二萬餘字粵拼庫，全在瀏覽器內離線運算——規則透明，即是可審計的非遺誠信。

平仄校驗 Verify

按行或標點分句。上句（單數）句末須仄；下句（雙數）句末須平且押同一韻。

⌘/Ctrl + Enter 平陰平/陽平 · 仄上去入聲

粵拼標註 Annotate

貼上任意中文（簡繁皆可），逐字標註粵拼、聲調與平仄，並顯示標音覆蓋率。

韻腳檢索 Rhyme

輸入一個漢字或粵拼（如 家 或 gaa1），檢索同韻母組的字，按平/仄分列。

句式模板 Template

選擇平仄模板，逐位比對你的句子是否合律。

跨代生成 Translate

將現代白話 / 流行歌詞，經 Claude 起草 → 平仄校驗 → 違規重寫 的神經-符號回路生成唱詞。

生成需調用大模型，密鑰不能置於公開頁面，故運行於 FastAPI 後端。 本工具台為公開靜態站，專注離線分析四件套。本地啟用： uvicorn app.main:app --reload 並設置 ANTHROPIC_API_KEY。
